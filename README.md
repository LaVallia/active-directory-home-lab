# Active Directory Home Lab

## Overview
Built a Windows Server 2019 Active Directory home lab using VirtualBox to simulate a corporate IAM/networking environment.

## Technologies Used
- Active Directory
- Windows Server 2019
- Windows 10
- Oracle VirtualBox
- PowerShell
- DHCP
- DNS
- NAT

## Skills Demonstrated
- User provisioning
- Access control
- DHCP configuration
- DNS troubleshooting
- Active Directory administration
- IAM fundamentals
- Infrastructure troubleshooting

## Lab Tasks Completed
- Configured Active Directory Domain Services
- Created Organizational Units (OUs)
- Bulk-provisioned domain users
- Configured DHCP services
- Joined Windows 10 client machine to domain
- Troubleshot DNS/domain communication issues
- Renamed and managed domain-joined client machine

## What I Learned
This project helped me better understand how authentication, DNS, DHCP, and Active Directory work together in enterprise environments. I also gained hands-on troubleshooting experience resolving domain communication and networking issues.
